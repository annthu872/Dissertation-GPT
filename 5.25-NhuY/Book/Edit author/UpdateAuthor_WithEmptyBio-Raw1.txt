Test Case: Test Case 1
testCaseName: Update Author with Empty Bio
objective: Verify that the book's author bio can be updated with an empty value
testSteps:
  - Step 1: Navigate to the book list/details page.
  - Step 2: Click the 'Update' button next to the book you want to update.
  - Step 3: In the form displayed, clear any existing data in the 'Author Bio' field.
  - Step 4: Fill in or confirm the remaining required fields with valid data.
  - Step 5: Press the 'Submit' button.
expectedResult: The book record is successfully updated in the database without an author bio, and the list of books with updated records is displayed.
explanation: This test case ensures that the system allows updating a book's author information with an empty biography, which is a valid scenario for authors without a detailed bio.

Test Case: Test Case 2
testCaseName: Cancel Update Operation with Empty Author Bio
objective: Verify that the update operation can be canceled after clearing the author bio
testSteps:
  - Step 1: Navigate to the book list/details page.
  - Step 2: Click the 'Update' button next to the book you want to update.
  - Step 3: In the form displayed, clear any existing data in the 'Author Bio' field.
  - Step 4: Start to fill in or confirm the remaining required fields.
  - Step 5: Press the 'Cancel' button before submitting the form.
expectedResult: The update operation is canceled, no data is updated in the database, and the list of book records is displayed to the user.
explanation: This test case checks the functionality of the cancel operation in the scenario where the author's bio is intended to be emptied but the update is canceled. It ensures that no changes are made to the database upon cancellation.
