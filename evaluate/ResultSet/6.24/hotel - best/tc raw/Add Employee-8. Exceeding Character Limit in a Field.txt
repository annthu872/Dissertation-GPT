Test Case: Test Case 1
testCaseName: Exceeding Character Limit in Employee Name Field
objective: Verify that the system displays an error message when the character limit is exceeded in the employee name field.
testSteps:
  - Step 1: Log in to the system with a manager account.
  - Step 2: Navigate to the Employee menu by clicking on 'Employee'.
  - Step 3: Fill all the fields, ensuring to exceed the character limit for the employee name field.
  - Step 4: Click on 'Add' to attempt adding the employee.
expectedResult: The system should display an error message indicating that the character limit for the employee name field has been exceeded.
explanation: This test case ensures that the system properly validates the character limit for the employee name field and provides appropriate feedback to the manager, preventing the addition of an employee with an excessively long name.
