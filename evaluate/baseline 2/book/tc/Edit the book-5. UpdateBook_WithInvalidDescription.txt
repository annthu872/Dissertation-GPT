Test Case: Test Case 1
testCaseName: Update Book with Invalid Description
objective: Verify that the system displays an error message when the user attempts to update a book with an invalid description
testSteps:
  - Step 1: Navigate to the book list/details page.
  - Step 2: Click the 'Update' button next to the book you wish to update.
  - Step 3: In the displayed form, enter or modify the Title, Author, Genre, and Publication date with valid data.
  - Step 4: Enter an invalid description (e.g., using special characters not allowed or exceeding the character limit).
  - Step 5: Press the 'Submit' button.
expectedResult: The system should display an error message indicating the description is invalid and prevent the book record from being updated.

Test Case: Test Case 2
testCaseName: Cancel Update Operation on a Book
objective: Verify that no changes are saved and the user is redirected back to the list of books when the 'Cancel' button is clicked during an update operation
testSteps:
  - Step 1: Navigate to the book list/details page.
  - Step 2: Click the 'Update' button next to the book you wish to update.
  - Step 3: Start entering or modifying the data in the form fields (Title, Author, Genre, Description, and Publication date).
  - Step 4: Press the 'Cancel' button before pressing 'Submit'.
expectedResult: The system should not save any changes and should redirect the user back to the list of books, displaying the original book records without any updates.

Test Case: Test Case 3
testCaseName: Successfully Update a Book with Valid Data
objective: Verify that the book record is successfully updated in the database and displayed in the list of books when all entered data is valid
testSteps:
  - Step 1: Navigate to the book list/details page.
  - Step 2: Click the 'Update' button next to the book you wish to update.
  - Step 3: Enter valid data for all fields (Title, Author, Genre, Description, and Publication date).
  - Step 4: Press the 'Submit' button.
expectedResult: The system should successfully update the book record in the database and redirect the user to the list of books, where the updated book details are displayed.
