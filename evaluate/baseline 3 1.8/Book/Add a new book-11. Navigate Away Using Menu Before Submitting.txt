Test Case: Test Case 1
testCaseName: Navigate Away Using Menu to 'Books' Before Submitting New Book Data
objective: Verify that navigating away to the 'Books' page using the menu before submitting new book data does not save the data
testSteps:
  - Step 1: Navigate to the book list page.
  - Step 2: Click the 'Create new' button to start adding a new book.
  - Step 3: Enter book data into the form fields (Title, Author, Genre, Description, Publication Date).
  - Step 4: Without pressing the 'Submit' button, select the 'Books' menu item to navigate away.
expectedResult: The application does not save the entered book data and displays the list of books records.
explanation: This test case ensures that if a user navigates away from the book creation page by selecting 'Books' from the menu before submitting the form, the entered data is not saved. It directly matches the scenario of navigating away using the menu before submitting.
