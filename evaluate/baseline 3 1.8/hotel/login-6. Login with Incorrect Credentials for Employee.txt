Test Case: Test Case 1
testCaseName: Login with Incorrect Employee Credentials
objective: Verify that an employee cannot log in with incorrect credentials and receives an appropriate error message.
testSteps:
  - Step 1: Navigate to the login page.
  - Step 2: Enter incorrect username and password for an employee account.
  - Step 3: Click the submit button.
expectedResult: The system should not log the user in and should redirect back to the login page with an error message indicating incorrect credentials.
explanation: This test case ensures that the system correctly handles login attempts with incorrect credentials by an employee, preventing unauthorized access and informing the user of the error.
