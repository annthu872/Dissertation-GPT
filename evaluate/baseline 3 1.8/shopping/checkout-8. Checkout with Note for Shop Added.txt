Test Case: Test Case 1
testCaseName: Checkout with Note for Shop Added
objective: Verify that the user can add a note for the shop during the checkout process and successfully place an order with the note included.
testSteps:
  - Step 1: Log into the Shopee account.
  - Step 2: Perform the 'Purchase' feature to initiate the checkout process.
  - Step 3: Review the order summary displayed by the system, including items, quantities, prices, and total cost.
  - Step 4: Confirm the shipping address is set as default or change/add a new shipping address if necessary.
  - Step 5: Confirm the payment method is set to COD or select a new payment method (e.g., Shopee Pay, Google Pay, credit).
  - Step 6: Optionally add an application voucher or shop voucher to the order.
  - Step 7: Confirm the shipping option is set to Standard Express or change the shipping option if desired.
  - Step 8: Add a note for the shop in the provided field.
  - Step 9: Review the updated total cost and order details.
  - Step 10: Click on the 'Place Order' button.
  - Step 11: Wait for the system to process the payment and confirm the order.
  - Step 12: Review the order confirmation and details displayed by the system.
expectedResult: The order is successfully placed with the note for the shop included, and the user receives an order confirmation with the note details.
explanation: This test case ensures that users can add notes for the shop during the checkout process, which is a part of enhancing the communication between buyers and sellers. It directly matches the scenario name by testing the ability to checkout with a note for the shop added.
