Test Case: Test Case 1
testCaseName: Successful Search with Brand Name Keyword
objective: Verify that the system displays the most related/best sold shop's information and related product cards when a brand name keyword is entered in the search bar.
testSteps:
  - Step 1: Navigate to the search bar on the application.
  - Step 2: Type a brand name keyword into the search bar.
  - Step 3: Press enter or click the search icon to initiate the search.
expectedResult: The system should display the most related/best sold shop's information that sells the brand at the top of the search result, including shop name, number of items, ratings, and response rate, average response time. Below the shop's information, related product cards (products with names matching the search keyword) should be displayed.
explanation: This test case is created to ensure that when users search using a brand name keyword, the system correctly identifies and displays the most relevant shop's information along with related product cards, as specified in the alternative flow for brand name keyword searches. It directly matches the scenario name by focusing on the successful search outcome when a brand name keyword is used.
