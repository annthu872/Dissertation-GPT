Test Case: Test Case 1
testCaseName: Navigate Away Using Menu Before Confirming Deletion to 'Books'
objective: Verify that navigating away to 'Books' from the deletion confirmation dialog cancels the deletion process and displays the 'Books' page.
testSteps:
  - Step 1: Navigate to the genre list/details page.
  - Step 2: Click the 'Delete' button next to the genre you wish to delete.
  - Step 3: On the deletion confirmation dialog, instead of clicking 'Confirm', select the 'Books' menu item.
expectedResult: The application cancels the deletion process, does not delete the genre, and navigates to the 'Books' page displaying updated data.
explanation: This test case ensures that when a user decides to navigate away using the menu before confirming deletion, the deletion process is aborted, and the user is redirected as per their menu selection. It directly tests the scenario of navigating away before confirming deletion, focusing on the 'Books' navigation.
